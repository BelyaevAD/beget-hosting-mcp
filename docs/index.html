Beget Hosting MCP Server

A Python MCP server for the official Beget hosting API. Use it from MCP-compatible clients to inspect and manage domains, DNS, sites, mail, MySQL, FTP, cron, backups and account data.

What It Provides

MCP Tools

Expose Beget API methods through standard MCP tool discovery and calls.

Guarded Writes

Write operations require both server-level enablement and per-call confirmation.

Local Docs

Includes Markdown and JSON copies of Beget hosting API documentation for agent use.

Supported Beget API Areas

  • Account information and limits
  • Domain and subdomain listing
  • DNS record inspection and controlled updates
  • Sites, FTP, MySQL, mail, cron, backups and statistics where the account API permits access
