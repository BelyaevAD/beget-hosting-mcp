# Beget.API Хостинга

Source: [https://beget.com/ru/kb/api/beget-api](https://beget.com/ru/kb/api/beget-api)
Обратите внимание!Эта документация описывает API хостинга.

[Документация API для наших облачных сервисов](https://developer.beget.com) доступна отдельно – там вы найдете информацию о том, как получить и обновить токен JWT и многое другое.

В статьях этого раздела мы рассмотрим, как пользоваться API для работы с хостингом Beget.

## **API – что это такое**

**API (Application Programming Interface)** – программный интерфейс или набор функций для управления сайтами, доменами, почтой и т. д.

![api хостинга](https://beget.com/img/slides/api@2x.png)

API по большей мере предназначен для программистов и разработчиков, однако им может воспользоваться любой желающий. Будьте внимательны, используя API для работы со своим действующим аккаунтом.

**Beget.API** позволит воспользоваться большинством функций панели управления из своих собственных скриптов, систем управления сайтами (CMS) или других программных продуктов.

## Что можно сделать, используя Beget.API

- Получить информацию о хостинг-аккаунте;

- Управлять резервными копиями;

- Управлять планировщиком заданий;

- Производить настройку DNS;

- Управлять базами данных;

- Создавать и удалять сайты на аккаунте;

- Управлять настройками доменов;

- Управлять почтовыми ящиками.

Наша команда всегда рада пожеланиям и отзывам клиентов о том, как работает API. Отправляйте ваши пожелания по работе с Beget.API на [manager@beget.com](mailto:manager@beget.com).
